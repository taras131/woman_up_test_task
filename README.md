## Инструкции по запуску

Удостовериться, что установлены все зависимости `npm i` и актуальная версия
Node.js. Тесты и сборка запускаются на `v17`.

- запуск 

```bash
npm run start
```

- билд

```bash
npm run build
```

- запуск eslint

```bash
npm run lint 
```

- запуск eslint--fix

```bash
npm run lint:fix 
```
